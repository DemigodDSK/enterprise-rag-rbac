# SOP: Data Access Approval
Company: RT Healthcare
Owner: IT & Compliance
Version: 2.0

---

# 1. Purpose

Defines controlled process for granting access to systems and data.

---

# 2. Access Request Process

1. User submits access request
2. Manager approval
3. Role validation
4. Security review (if privileged)
5. Provisioning
6. Documentation in access log

---

# 3. Access Types

- Standard user access
- Elevated/privileged access
- Temporary access
- Vendor access

---

# 4. Access Reviews

- Quarterly review
- Immediate revocation upon termination
- Annual privileged access audit

---

# 5. Logging

All access provisioning must be logged and auditable.
